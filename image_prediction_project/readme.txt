Questa repository contiene solamente i file relativi alla creazione dei modelli con relative stampe riguardo le accuratezze ottenute tramite k-fold cross validation.
Il dataset contenente le immagini, come descritto nel file di preprocessing,è stato trasformato in un file "npy" per poter essere letto dai modelli velocemente. 
Ogni modello è stato poi salvato ma non inserito all'interno della cartella github a causa della memoria occupata.
